# Bank_Loan-Portfolio-Health-Analysis-PowerBI
Interactive Power BI dashboard for Loan Portfolio Health Analysis featuring Executive Summary, Loan Analysis, Risk Analysis, and Customer Analysis using DAX, Power Query, SQL, and Star Schema data modeling.

Loan Portfolio Health Analysis Dashboard | Power BI

This project is an end-to-end Business Intelligence solution developed in Power BI to analyze the overall health and performance of a bank's loan portfolio. The dashboard transforms raw loan data into actionable insights, enabling stakeholders to monitor lending performance, identify credit risk, and understand customer behavior through interactive visualizations.

The solution consists of four analytical dashboards:

Executive Summary – Portfolio KPIs and overall business performance
Loan Analysis – Loan distribution by grade, purpose, state, term, and monthly trends
Risk Analysis – Default rate, recovery analysis, portfolio risk, and loan performance
Customer Analysis – Customer segmentation, income analysis, employment trends, and verification status

The project demonstrates the complete Business Intelligence workflow, including data cleaning, transformation, data modeling, DAX calculations, KPI development, dashboard design, and business storytelling.
